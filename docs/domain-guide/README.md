# HeatGrid Domain Guide

HeatGrid Domain Guide는 `PreDist 기반 지역난방 운영 AIoT Agent`를 설계하기 위한 초심자용 도메인 학습 제품군이다.

## 구성

- `00_HeatGrid_Domain_Guide.md`
  - 전체 학습 로드맵
- `01_*.md` ~ `12_*.md`
  - 핵심 논문, PDF, 국내 자료 요약집
- `assets/pdf/`
  - 로컬에 저장한 원문 PDF
- `index.html`
  - GitHub Pages 허브
- `sources_manifest.md`
  - 자료 원문 링크와 로컬 보관 상태

## 권장 학습 순서

1. `00_HeatGrid_Domain_Guide.md`
2. `01_PreDist_논문_정리.md`
3. `02_PreDist_데이터셋_가이드.md`
4. `03_국내_지역난방_구조와_운영_가이드.md`
5. `04_해외_지역난방_구조와_운영_가이드.md`
6. `05` ~ `10` 문서
7. `11`, `12` 문서

## 문서 작성 규칙

- 모든 문서는 한국어 중심으로 작성한다.
- 영어 원문은 핵심 문장 번역과 해설 위주로 정리한다.
- 각 문서에는 원문 링크와 로컬 PDF 경로를 함께 적는다.
- 각 문서에는 `읽는 시간`, `난이도`, `선수지식`을 표기한다.
