08. District Energy PM Checklist 요약

중요한 번역 포인트

  • 열교환기 연결부 누수 점검
  • 3-way valve 동작 점검
  • 계측기 정확도 확인 및 보정
  • 열교환기 세정 주기 관리
  • 안전 릴리프 밸브 점검

HeatGrid에 직접 쓰는 포인트

  • Agent 출력은 이상 유형 -> 체크리스트 매핑이 있어야 한다.
  • 압력 이상, 온도 이상, 밸브 이상마다 다른 현장 확인 순서가 필요하다.

초심자 체크포인트

  • 숫자 경보를 현장 체크리스트로 바꿀 수 있는가
