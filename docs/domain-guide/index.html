HeatGrid Domain Guide

지역난방을 몰라도
HeatGrid Agent 설계까지 갈 수 있는 학습 사이트

이 사이트는 지역난방, 기계실, 정비, 운영, PreDist를 처음 보는 사람도 문제 이해 → 데이터 해석 → 운영 판단 → Agent 설계까지 연결하도록 만든 도메인 가이드입니다.

처음 시작하기 국내 구조와 운영 먼저 보기 해외 구조와 운영 먼저 보기

문제

지역난방 기계실은 구조도 낯설고, 센서와 부품, 민원과 출동, 예방정비와 재계획이 한 번에 얽혀 있어 처음 보면 무엇부터 공부해야 할지 감이 잘 오지 않습니다.

해결

이 사이트는 PreDist, 국내 운영 자료, 해외 O&M 문서, 정비 체크리스트를 한 흐름으로 엮어 초심자도 단계적으로 이해할 수 있게 정리합니다.

결과

다 읽고 나면 주요 부품, 센서 변수, 대표 고장, 작업지시, 우선순위화, 재계획 구조를 설명하고 HeatGrid Agent 설계에 연결할 수 있습니다.

아래 순서로 보면 도메인이 없는 사람도 자연스럽게 올라올 수 있습니다.

전체 문서

  • 전체 로드맵도메인 초심자가 어디서부터 무엇을 배우고 어떻게 Agent 설계까지 연결할지 안내합니다.
  • PreDist 논문데이터셋 배경, 라벨 구조, 리드타임 개념을 가장 빠르게 이해합니다.
  • PreDist 데이터셋실제 파일 구조와 멀티 기계실 데이터를 HeatGrid 관점으로 읽는 방법을 정리합니다.
  • 국내 구조와 운영한국의 집단에너지 구조, 민원/긴급신고, 정기점검, 협력업체 운영 흐름을 설명합니다.
  • 해외 구조와 운영ETS/substation, 예방정비, 체크리스트, O&M 문화 등 해외 운영 관행을 정리합니다.
  • 기계실 설계열교환기, 밸브, 펌프, 센서 등 기계실 구성요소와 정비 분류를 이해합니다.
  • 현장 O&M실제 점검 항목과 예방정비 주기를 HeatGrid 작업지시 관점으로 해석합니다.
  • 운영 기록과 PM기록, 정비 일정, 연결 구조가 왜 운영 계획의 기반인지 설명합니다.
  • 체크리스트정비사가 현장에서 확인하는 체크 포인트를 바로 읽을 수 있게 정리합니다.
  • 유지관리 계획inspection, maintenance plan, condition indicator 같은 핵심 운영 개념을 잡습니다.
  • 이상탐지 연구부분 고장, 성능 저하, 운영 리스크를 왜 anomaly 관점으로 봐야 하는지 이해합니다.
  • 국내 시장과 정책누가 구매자고 사용자며, 왜 지금 한국에서 이 서비스가 맞는지 설명합니다.
  • 정비사와 출동현장 정비사가 실제로 어떤 정보를 갖고 움직이는지 이해합니다.
  • 폴더 안내Domain Guide 폴더 구조와 읽는 순서를 빠르게 확인합니다.
