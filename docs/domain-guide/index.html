HeatGrid Domain Guide

지역난방을 몰라도
HeatGrid Agent 설계까지 갈 수 있는 학습 사이트

시간이 없다면 "00 이 문서 하나로 충분합니다" 한 편만 읽어도 지역난방 기계실부터 HeatGrid Agent까지 한 번에 이해할 수 있다. 더 깊이 파보고 싶을 때만 나머지 문서로 넘어가면 된다.

이 문서 하나로 시작하기 국내 구조와 운영 먼저 보기 해외 구조와 운영 먼저 보기

문제

지역난방 기계실은 설비 구조, 센서 데이터, 민원, 정비, 출동, 우선순위화가 한 번에 얽혀 있다. 처음 보면 어디서부터 공부해야 할지 감이 잘 안 잡힌다.

해결

이 사이트는 PreDist, 국내 운영 자료, 해외 O&M 문서, 정비 체크리스트를 한 흐름으로 엮어 초심자도 단계적으로 이해하도록 구성했다.

결과

끝까지 읽고 나면 기계실 구조, 주요 고장, 작업지시, 운영 우선순위, Agent 필요성을 자기 말로 설명할 수 있다.

학습 순서

아래 순서대로 보면 초심자도 자연스럽게 설비 구조에서 Agent 설계까지 이어진다.

  1. 00 이 문서 하나로 전체 그림 잡기 (다른 문서 없이도 충분)
  2. 01~02 PreDist와 데이터 구조 이해
  3. 03~04 국내와 해외 지역난방 구조 비교
  4. 05~10 기계실 설계, O&M, 체크리스트, 이상탐지 해석 심화
  5. 11~12 시장, 정책, 정비사 출동 프로세스로 서비스화 연결

핵심 진입점
