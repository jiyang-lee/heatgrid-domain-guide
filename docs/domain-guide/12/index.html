12. 정비사 업무와 출동 프로세스 가이드

정비사의 일반적인 흐름

  1. 이벤트를 받음
  2. 현장 이동
  3. 계측/관찰
  4. 원인 후보 축소
  5. 조치 수행
  6. 결과 기록
  7. 재발 여부 확인

현장 출동에 필요한 정보

  • 위치/설비 ID
  • 현재 이상 징후
  • 우선 점검 항목
  • 예상 원인
  • 준비 부품/도구
  • 안전 주의사항
  • 과거 작업 이력

HeatGrid에 직접 쓰는 포인트

  • Agent는 현장 수리 자체를 대신하지 않는다.
  • 대신 무엇을 먼저 보낼지, 어떤 정보를 들고 갈지, 계획이 틀어지면 어떻게 바꿀지를 돕는다.

초심자 체크포인트

  • 좋은 작업지시서에 어떤 정보가 들어가야 하는지 설명할 수 있는가
  • 정비사 관점에서 Agent의 가치가 무엇인지 설명할 수 있는가
