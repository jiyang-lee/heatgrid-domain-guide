# HeatGrid Domain Guide

HeatGrid Domain Guide는 `PreDist 기반 지역난방 운영 AIoT Agent`를 설계하기 위한 초심자용 도메인 학습 사이트입니다.

## 로컬 사이트 진입점

- [docs/index.html](docs/index.html)
- [docs/domain-guide/index.html](docs/domain-guide/index.html)

## 주요 구성

- `docs/domain-guide/index.html`
  - GitHub Pages 랜딩 페이지
- `docs/domain-guide/00` ~ `12`
  - 개별 HTML 학습 문서
- `docs/domain-guide/assets/pdf`
  - 원문 PDF 로컬 자산
- `docs/domain-guide/references`
  - HeatGrid 기획 문서 HTML 참조본

## 목적

도메인 지식이 없는 사람도 이 사이트만 따라가면 아래를 설명할 수 있도록 설계했다.

- 지역난방 기계실 구조
- PreDist 데이터 해석
- 주요 고장과 점검 항목
- 운영 우선순위와 재계획
- HeatGrid Agent 설계 연결
